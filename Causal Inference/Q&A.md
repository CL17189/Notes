2. 专业选择思路：城市&学校&专业之间的考虑
+ 城市会影响的因素：
	+ 气候（南北方&潮湿与否）
	+ 环境（城市的基础建设，比如公共交通的便利程度）
	+ 兄弟院校之间的交流机会（如讲座；跟某个外校老师做科研的机会（省去通勤的麻烦））
	+ 经济发达程度（会影响就业、实习等）
	+ 某行业发达程度（会影响就业、实习等）
	+ 后两条的具体逻辑是这样的：城市的经济发达程度会决定城市的产业结构，某行业的发达程度同样也会决定某个产业在该城市的发达程度。比如上海可以有很多外企招实习生而北京比较少，但是北京的互联网行业的赛道（细分产业）是最全的；对不搞学术的当你决定把某个领域作为职业深耕的时候，这种机会是必要的，不然可能要去选择花费时间和金钱做某漂；
+ 学校会影响的因素：
	+ 标签（如985等，会是参加某种筛选的标签并且竞争越激烈被参考的概率越大）
	+ 学校拿到的学术资源（看经费、看学校拿到的奖和项目等等）
+ 专业会影响的因素：
	+ 培养方案
		+ 这个非常重要。该学校该专业的培养方案是不是“合理的”？培养方案可以私聊招生组获取。尽管内地几乎所有的大学培养方案都是不正确的，但是可以参考的几个方向：是否涵盖了大部分的专业课（以供考研使用）、是否有不合理占用课余时间的任务（强制实习等——个人没收获，耗费时间长）、没用的课程是不是可以水（如思政）；这一条可以问招生组培养方案的细节，或者直接上学校论坛问（百度贴吧搜索也可））
		+ 没有那么多占用课余时间的不合理事情的好处是大量可以自由安排的时间。如想玩，可以充分的玩，如想卷也可以充分的卷。
	+ 学校给学院分配的资源（主要是一些交流的机会）
	+ 学院给个人分配的资源（指的是能落实到个人的资源，比如交换的机会是不是只是被年级最前面的同学拿走了。可以问学校论坛）
	+ 朋辈之间传递的资源（比如内推实习、科研等）
	+ 教学水平（专业课老师是不是在念10年前做的ppt？）


2. 心理学的出路
	+ 临床、教职
		+ 这一个方向很好理解。心理医师或者咨询师。需要学历和大量的经验
	+ 互联网、咨询公司等可以和心理学结合的方向
		+ 注意：当你选择了这个方向，其实就是选择了某些商科的方向，你的心理学背景只是可能会在日常工作中发挥作用。
		+ 为什么这个方向是可行的？->可以去某些招聘软件上看，不少咨询等职位是欢迎心理学背景的同学的。
		+ 职位举例：用户研究、某些运营岗、某些数据分析岗和战略分析岗等等
		+ ![[41d2981a2daea9a0ff8852e884626dc.jpg]]
		+ 职业规划路径：商科的同学需要丰富的实习经历，而影响实习经历的因素可能是城市、城市特点、朋辈之间的资源。实习经历可以靠自己争取（需要自己有比较好的实力和经历符合程度，直接去那些软件上投递；但是院校分有时候比较重要，所以学校有时候会是比较大的影响因素，这个需要具体看公司。但是大的趋势是由于经济不太好竞争在逐渐变激烈）、也可以靠朋辈内推（需要建立比较好的社交网络，能吃到资源）；同时如果是走这个方向需要自学一部分行业相关内容，可以从小红书、b站等地方找课自学
	+ 总结：心理学“正统”的就业方向职位供给比较少，尤其国内现在还不是很重视普通的咨询师；而教职和医师又需要非常高的学历（博士）；去商科方向会比较卷并且需要自己在学校的培养方案外做非常多的事情，但是也是一个不错的出路；
